# 자바스크립트
## _Front-End_

- ✨객체 기반 언어
- 모든 웹 브라우저에서 작동
- 다양한 라이브러리와 프레임워크 사용가능

## 규칙
- 대소문자 구별하여 소스 작성
- 세미콜론으로 문장 구분
- 식별자는 정해진 규칙 지켜 작성
- 예약어는 식별자로 사용불가


## 자료형
- 자바스크립트는 미리 변수의 자료형을 지정하지 않음
- 변수를 지정하고 원하는 값을 할당만 하면 됨

document : 문서 객체
document.write() : 데이터를 문서에 출력할 때 사용
console.log() : 콘솔에 출력하는 것 (프로그래머 전용..이라고 할 수 있음. 실제로 프로그램 완성할 땐 주석처리)

! + tab -> 뼈대 자동 생성
