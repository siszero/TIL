# 조건에 맞는 데이터 추출

- a 가 3 이상인 데이터 출력
```
df[df['a'] >= 3]
```

- a 가 3 이상인 데이터 중 a, c 열만 출력
```
df[df['a'] >= 3][['a', 'c']]
```

- a 가 3 이상이고, b 가 16 미만인 데이터 출력
```
df[(df['a'] >= 3) & (df['b'] < 16)]

# 아니면 a에 넣어주고 a를 출력해주기
a = (df['a'] >= 3) & (df['b'] < 16)
df[a]
```

- a 가 3 이하 이거나 7 이상인 데이터 출력
```
df[(df['a'] <= 3) | (df['a'] >= 7)]
```

- a 가 3 이상이고, b 가 16 미만이거나 c 가 30 인 데이터를 출력
```
df[(df['a'] >= 3) & ((df['b'] < 16) | (df['c'] == 30))]
```
