```

```

**numpy 라이브러리는 행렬, 선형대수, 딥러닝에서 자주 쓰임**

```
import numpy as np          # numpy를 np라고 별명붙임
b = {'company' : ['abc', '회사', '123'], '직원수' : ['400', '10', '6'], '위치' : ['Seoul', np.NaN , 'Busan']}
df1 = pd.DataFrame(b)
df1
```
