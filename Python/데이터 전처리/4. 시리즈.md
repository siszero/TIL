# 시리즈

**데이터프레임['해당 필드명']**
- 한 줄 씩 출력
- 리스트와 다른점 : 인덱싱 번호가 앞에 주어진다. (인덱스 변경 가능)


### 시리즈 생성 방법
```
import pandas as pd

df = pd.DataFrame({'a': [1, 2, 3], 'b' : [4, 5, 6], 'c' : [7, 8, 9]})
```
```
df['a']         # a를 시리즈로 만들기
```

- 인덱스 변경 (시리즈 개수와 인덱스 개수 같아야 함)

- 유일한 값 찾기
**.unique()**
문제 ) 아래 테이블(df) 의 a 필드의 값들 중 유일한 것들만 추출하시오.
```
df = pd.DataFrame({'a': [1, 2, 3, 1, 2, 3], 'b' : [4, 5, 6, 6, 7, 8], 'c' : [7, 8, 9, 10, 11, 12]})
a = df['a']
```
```
a.unique()
```
실행결과   
array([1, 2, 3])   

```
a.unique()[2]
```
실행결과   
3
