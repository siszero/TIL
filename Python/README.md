# Python

## 문법
- 세미클론(;) 으로 끝을 맺지 않아도 된다.
- 들여쓰기에 주의가 필요하다.
- 주석은 #(한 줄) 또는 ''', """(따옴표 세개 - 여러 줄) 로 만들 수 있다.

## 변수
- 변수명 작성시 주의사항
    - 영문자, 숫자, _ 의 조합으로 구성되어야 함
    - 변수명의 첫 글자는 숫자로 사용할 수 없음
    - 대문자와 소문자는 구분되어 사용됨
    - 변수명 사이에 띄어쓰기는 불가

```
a = 100
b = 10000
c, d = 100, 1000
print(a)    # a 값 출력
print(b / a)    # b / a = 100 출력
```
