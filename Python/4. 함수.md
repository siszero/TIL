# 함수

- 함수를 만드는 이유 : 똑같은 작업을 효율적으로 하기 위해
- 생성방법
```
def 함수명(인풋 값):
  실행 내용
```

### 문제
- x 를 입력값으로 넣으면 x + 1 을 출력하는 함수 생성
```
def function1(x):
    print(x + 1)

function1(1)            # 2
```

- x 를 입력값으로 넣으면 x + 1 을 반환하는 함수 생성
```
def function2(x):
    result = x + 1
    return result

a = function2(1)
print(a)                # 2
```


- x 를 입력값으로 넣으면 x + 1, x + 2 를 반환하는 함수 생성
```
def function3(x):
    result1 = x + 1
    result2 = x + 2
    return result1, result2

a, b = function3(1)
print(a)                # 2
print(b)                # 3
```

- 여러개의 입력값을 받고 싶을 때
```
# 여러개의 입력값을 받아 순서대로 출력
def function4(x):
    for i in x:
        print(i)

a = [1, 3, 5, 7, 4, 'd', 2, 60]
function4(a)
```

```
# args 를 이용한 구현
def function5(*args):
    for i in args:
        print(i)

function5(1, 3, 5, 7, 4, 'd', 2, 60)
```
